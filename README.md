| **CI/CD** | GitHub Actions로 MSA 모듈별 워크플로우 구성, 변경된 모듈 Docker 이미지 빌드 및 자동 배포 |
| **Deploy** | Docker · Nginx 기반 서비스 배포 및 운영 |
| **Management** | WBS · 화면정의서 · Git Issue · 기능별 브랜치 전략으로 프로젝트 관리 |
